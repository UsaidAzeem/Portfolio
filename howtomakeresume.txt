🧱 1. Overall Layout Structure (VERY IMPORTANT)

This resume uses a 2-column asymmetric grid:

| LEFT COLUMN (thin) | RIGHT COLUMN (wide) |
Left column (~30–35%)
Personal info
Skills
Languages
Interests
Right column (~65–70%)
Name + Title
Experience
Education
Awards / Recognition

👉 Think of it as:

display: grid;
grid-template-columns: 1fr 2fr;
gap: medium;
🎯 2. Visual Style (Minimalist Swiss Design)

This resume follows a Swiss / editorial design style:

Black text on white background
No borders, no boxes
Heavy reliance on spacing and alignment
Clean sans-serif font (likely similar to):
Helvetica / Inter / Neue Haas Grotesk
Typography hierarchy:
Element	Style
Name	LARGE, bold
Section titles	small caps / uppercase
Body text	small, light
Dates	subtle, aligned

🧱 5. RIGHT COLUMN CONTENT

This is where most structure complexity is.

💼 Experience Section

Each job follows a VERY specific pattern:

Structure:

[Location]
[Role]

[Date]

[Description paragraph]
Example pattern:

Key Design Details:
Company name → slightly bold
Location → lighter
Role → same size as company
Date → separate line (important spacing)
Description → paragraph (NOT bullets)

👉 Spacing:

Tight between company/location/role
Medium gap before date
Small gap before description
BIG gap before next job

📏 6. Spacing System (CRITICAL)

This resume works because of spacing consistency:

Use a spacing scale like:
XS → 4px
SM → 8px
MD → 16px
LG → 32px
XL → 48px
Apply like:
Between sections → LG/XL
Between items → SM/MD
Between title + content → MD
🔠 7. Alignment Rules
EVERYTHING is left-aligned
No center alignment anywhere



This style is clearly based on Swiss modernist typography, meaning fonts like:

Helvetica Neue (most likely inspiration)
Inter (best modern free match)
